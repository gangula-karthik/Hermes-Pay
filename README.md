# Hermes Pay 🕊️

The solution to help PWIDs make purchases at F&B establishments easily. <br>
There's a demo video if you're curious to learn more: https://www.youtube.com/watch?v=InLpe0j8oSU

## Tech Stack 🛠️
- NextJS
- Typescript
- Python
- Flask
- Ollama
- PaddleOCR
